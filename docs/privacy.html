Privacy

Local by design.

fan does not need your identity or your data to manage cooling.

What fan reads

fan reads thermal sensors, fan speeds, available battery power information, and its own saved preferences. These values remain on the Mac.

What fan sends

Nothing. fan has no account system, advertising SDK, analytics service, crash uploader, or telemetry endpoint.

Diagnostics

A diagnostics export is created only when the user chooses Export Diagnostics. The file stays local until the user decides to share it.

Website

This static GitHub Pages site does not set application cookies or run first-party analytics. GitHub may process standard request data under its own privacy statement.
