# fan 1.0

Released 12 July 2026.

fan 1.0 is a fresh public release of the native macOS menu bar cooling utility. It combines live thermal information with clear, reversible fan controls in a compact interface.

## Highlights

- A redesigned thermal cockpit with temperature, fan RPM, power draw, and fan count.
- Five cooling profiles: System, Quiet, Balanced, Performance, and Max.
- Smart control with configurable cooling threshold and response.
- Manual RPM control for predictable sustained cooling.
- A 10 minute boost for short, intensive workloads.
- Menu bar display modes for temperature, power, fan percentage, or icon only.
- Adaptive sensor polling and serialized SMC access.
- Emergency maximum-speed protection.
- Automatic restoration on exit, sleep, lock, and abnormal termination.
- Diagnostics export for support and compatibility reports.

## Installation

1. Download `fan-1.0.0-macos.dmg`.
2. Drag `fan.app` to Applications.
3. Right-click the app and choose Open on first launch.
4. Select Install helper and approve the administrator prompt once.

The supplied v1.0 build targets Apple Silicon and requires macOS 26.1 or later.

## Security and privacy

fan performs thermal monitoring and control locally. It contains no account system, analytics, advertising SDK, or telemetry upload.

Fan-speed changes require a narrowly scoped helper at `/usr/local/bin/smc-helper`. The main application does not run as root. Review [SECURITY.md](../../SECURITY.md) for details.

## Known limitations

- Sensor names and fan capabilities differ between Mac models.
- Some Apple Silicon models expose fewer SMC keys than others.
- The release is ad-hoc signed, so macOS may require Right-click, Open on first launch.
- The supplied v1.0 download is Apple Silicon only.
