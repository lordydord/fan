# Fan App 1.1.3

Fan App 1.1.3 corrects how the Smart control response setting affects fan speed.

## What changed

- Maximum response no longer forces maximum RPM while the Mac is below its target temperature.
- At or below the target, every response level now returns the fans toward their minimum speed.
- Above the target, stronger response settings ramp the fans earlier and harder than gentler settings.
- The separate Max profile still commands full fan speed continuously.
- Emergency protection still commands full speed when its safety temperature is reached.

## Example

With Smart control set to a 55°C target and Maximum response:

- At 55°C, the verified fan speed returned to approximately 1,001 RPM.
- When the target was temporarily lowered to 40°C while the Mac was at 54°C, the fans rose to approximately 4,887 RPM.
- Restoring the 55°C target brought the fans back down again.

This means Maximum now describes how strongly Fan App reacts above the target, rather than acting as a permanent full-speed command.

## Install

1. Download `fan-1.1.3-macos.dmg`.
2. Open the disk image and drag `fan.app` into Applications.
3. Right-click `fan.app` and choose **Open** on first launch.
4. Choose **Install helper** and approve the narrowly scoped fan-control helper when prompted.

The supplied build targets Apple Silicon and macOS 26.1 or later. It is ad-hoc signed, so macOS may display the standard warning for independently distributed apps.

SHA-256 checksum files are attached beside both downloads.
