# Fan App 1.1.4

Fan App 1.1.4 refreshes the complete menu bar experience with native macOS Liquid Glass while keeping the app compact, familiar, and focused on cooling.

## What changed

- The main dashboard and Settings now use native glass surfaces designed for macOS 26.
- Amber and temperature-aware tints remain subtle so live readings and controls stay easy to read.
- Profile controls, metric chips, action buttons, state panels, and settings sections now share one consistent visual system.
- The background is clearer and less milky, allowing the system popover material to provide the surrounding depth.
- Settings still opens inside the menu bar popover, and Back returns directly to the dashboard.

Cooling policies, Smart response curves, manual control, Max mode, helper behaviour, and emergency protection are unchanged in this visual update.

## Install

1. Download `fan-1.1.4-macos.dmg`.
2. Open the disk image and drag `fan.app` into Applications.
3. Right-click `fan.app` and choose **Open** on first launch.
4. Choose **Install helper** and approve the narrowly scoped fan-control helper when prompted.

The supplied build targets Apple Silicon and macOS 26.1 or later. It is ad-hoc signed, so macOS may display the standard warning for independently distributed apps.

SHA-256 checksum files are attached beside both downloads.
