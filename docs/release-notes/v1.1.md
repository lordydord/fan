# Fan App 1.1

Fan App 1.1 brings the settings experience into the same menu bar popover as the cooling dashboard and makes the whole app feel like one coherent product.

## Highlights

- Fan App branding throughout the live interface.
- Settings now replaces the dashboard inside the existing popover instead of opening another window.
- A consistent amber, translucent visual system across dashboard and settings.
- A smoother crossfade between dashboard and settings, with complete reduced-motion support.
- Max now commands every detected fan to the configured 6,500 RPM ceiling.
- Selected profiles persist across restarts, while manual adjustments are clearly identified as Custom.
- Regression coverage for maximum targets on one-fan and two-fan Macs.

## Install

1. Download `fan-1.1-macos.dmg`.
2. Open the disk image and drag `fan.app` into Applications.
3. Right-click `fan.app` and choose **Open** on first launch.
4. Choose **Install helper** and approve the narrowly scoped fan-control helper when prompted.

The supplied build targets Apple Silicon and macOS 26.1 or later. It is ad-hoc signed, so macOS may display the standard warning for independently distributed apps.

## Safety

Fan App clamps targets to supported limits and restores macOS automatic fan control on normal exit, sleep, lock, or abnormal process termination. The Max profile affects every fan reported by the Mac.

SHA-256 checksum files are attached beside both downloads.
